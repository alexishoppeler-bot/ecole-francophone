# Ecole Francophone

Site statique de cours et d'exercices de francais.

## Structure

- `index.html` : page d'accueil
- `cours/` : pages HTML des cours, entrainements et annexes
- `css/` : styles communs
- `js/` : logique du site
- `assets/` : images, audio, polices
- `docs/` : guides de modification
